# ccusage-compact

[ccusage](https://github.com/ryoppippi/ccusage) の `blocks` コマンドの出力を、80文字幅のコンパクトなテーブルで表示するスクリプト。

## 背景

`npx ccusage@latest blocks --live` は約110文字幅を必要とするため、狭い端末ではTokens列などが切れて表示される。本スクリプトは同等の情報を80文字幅に収めて表示する。

## 必要環境

- Node.js
- npx（ccusageの取得に使用）

外部npmパッケージへの依存なし。

## 使い方

### 1回実行

```bash
node ccusage-compact.mjs
```

### ライブ更新（30秒ごと）

```bash
node ccusage-compact.mjs --live
```

### ライブ更新（間隔を指定）

```bash
node ccusage-compact.mjs --live --interval 60
```

`--interval` の単位は秒。Ctrl+C で終了。

## カラムの見方

| カラム | 説明 |
|--------|------|
| Date (JST) | ブロック開始時刻（JST、`MM-DD HH:mm` 形式） |
| Status | 経過時間 / `ACTIVE` / `gap Xh` |
| Tokens | 合計トークン数（`438k` / `2.5M`） |
| % | トークン上限に対する割合（全ブロック最大値を上限として推定） |
| Cost | コスト（USD） |
| Models | 使用モデルの略称（カンマ区切り） |

### Statusの値

| 値 | 意味 |
|----|------|
| `35m` / `2h7m` | 完了したブロックの継続時間 |
| `ACTIVE` | 現在進行中のブロック |
| `gap 3h` / `gap 1h13m` | セッション間の空き時間 |

### Modelsの略称

| 略称 | モデル |
|------|--------|
| `son` | claude-sonnet-* |
| `hku` | claude-haiku-* |
| `opu` | claude-opus-* |
| `syn` | \<synthetic\> |

### REMAINING / PROJECTED 行

アクティブなブロックがある場合に表示:

| 行 | 内容 |
|----|------|
| REMAINING | 推定トークン上限の残量と割合 |
| PROJECTED | 現在の消費レートで5時間ブロックを使い切ったときの推定トークン数・コスト |
